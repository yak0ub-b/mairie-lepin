# Strasbourg.eu — Analyse design

## Navigation
- Navigation par **profils utilisateurs** : Association, Candidat, Entreprise, Jeunes, Journaliste, Parent, Nouvel arrivant, Touriste
- Secondaire : Travailler à l'Eurométropole / Démarches / Agenda / StrasApp / Vitaboucle
- Mon quartier / Ma commune / Contact / MonStrasbourg.eu

## Hero
- Bloc alerte prioritaire (ex: "Activation du plan pollution")
- CTA : "Retrouvez toutes les mesures mises en œuvre"
- Image secondaire avec titre article

## Couleurs (CSS Variables complètes)
- **Fond principal** : `--main: #171C22` (gris-bleu très foncé — mode sombre !)
- **Primaire/accent** : `--secondary: #2AD783` (vert émeraude)
- **Accent chaud** : `--tertiary: #FFD300` (jaune doré)
- **Texte** : `--textColor: #B1B9C3` (gris clair sur fond sombre)
- **Texte actif** : `--whiteColor: #FFFFFF`
- **Header** : `--headerBgColor: #20272F`
- **Border** : `--border: #51555A`

## Typographies
- `--f-hx: 'TTFors', Sans-Serif` (police propriétaire pour titres)
- `--f-txt: 'TTFors', Sans-Serif` (corps)
- Poids : h1 → 600, h3/h4/h5 → 700, texte → 300
- Pas de Google Fonts

## Sections homepage (ordre)
1. Header logo + nav
2. Bloc alerte pollution (contextuel)
3. NewsHome (actualité principale + image)
4. "L'actu" (8 articles vignettés)
5. Agenda (6 événements)
6. Éditions (magazines numérisés)
7. Dossiers thématiques (7)
8. Galaxie des sites (20+ liens)
9. Sélecteur territoires
10. Footer avec contacts + 8 réseaux sociaux

## Features notables
- ✅ **Multilingue** : 10 langues (fr, en, es, de, zh, ja, ru, it, nl, lsf)
- ✅ Recherche dynamique avec filtres
- ✅ Analytics Matomo + événements DDM
- ✅ 8 plateformes sociales intégrées
- ✅ Accessibilité RGAA (partiellement conforme)
- ❌ Pas de chatbot visible

## Points faibles
- **Design très sombre** : contraste texte `#B1B9C3` sur `#171C22` limite lisibilité
- Hiérarchie dense : 20+ sections
- Vignettes héros trop petites (300x300px) — peu impactantes
- Police TTFors propriétaire → risque cross-device
- CTA peu contrastés (vert sur gris foncé)

## Note globale
7/10 — Unique visuellement (mode sombre + vert émeraude), navigation profils très pertinente, multilingue exemplaire. Mais trop sombre pour un usage grand public municipal.
