# Rennes Métropole — Analyse design

## Navigation
- 5 items : Vos services / Vos institutions / Votre commune / Votre quartier / Recherche
- "Vos services" : 11 sous-catégories thématiques
- Navigation claire, peu de niveaux

## Hero
- Image full-width avec overlay
- CTA : "Accès rapides" (6 liens directs vers services clés)
- Message : "Bienvenue à Rennes Ville et Métropole"

## Couleurs
- Logo : noir (rennes-ville-metropole_noir.svg) + blanc footer
- Palette non visible dans CSS inline (externalisé)

## Typographies
- Non détectées dans le HTML fourni

## Sections homepage (ordre)
1. Navigation / Header
2. Hero image
3. **Accès rapides** (6 liens)
4. "Vous êtes" (3 personas utilisateurs)
5. "ICI Rennes Métropole Actualités"
6. "Nos autres sites thématiques" (5 portails)
7. Footer multi-colonnes

## Features notables
- ✅ Recherche intégrée
- ✅ Open Data (portail)
- ✅ Réseaux sociaux (4 canaux)
- ✅ **Éco-conception affichée** (argument différenciant rare)
- ✅ Navigation par personas ("Vous êtes")
- ❌ Pas de chatbot

## Points faibles
- Manque de hiérarchie visuelle claire
- Navigation dense sans distinction CTA primaire/secondaire
- Palette non distinctive (noir + blanc → très sobre)
- Peu d'identité locale visible

## Note globale
6/10 — Structure sobre et bien pensée, navigation personas intéressante, mais peu mémorable visuellement. L'éco-conception est un vrai argument.
