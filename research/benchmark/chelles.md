# Chelles.fr — Analyse design

## Navigation
- 4 items : Ma ville / Mes services / Mon temps libre / Mes démarches
- Navigation "Mes" → appropriation citoyenne (ton proche)
- Recherche intégrée + toggle mobile

## Hero
- Image de fond full-width (photo aérienne mairie)
- Titre "Bienvenue à Chelles" + sous-titre
- **CTA** : accès rapides sous la bannière (Culture, Emploi, Infos travaux, Démarches en ligne)

## Couleurs
- **Primaire** : `#0170B9` (bleu)
- **Secondaire** : `#3a3a3a` (gris foncé)
- **Accent** : `#4B4F58` (gris moyen)
- **Fond** : `#F5F5F5` (gris très clair)
- **Blanc** : `#FFFFFF`

## Typographies
- `-apple-system, BlinkMacSystemFont, Segoe UI, Roboto` (polices système)
- Poids 600 pour titres
- Pas de Google Fonts

## Sections homepage (ordre)
1. Hero + CTA rapides
2. Actualités/À la une (3 articles)
3. Agenda événementiel avec filtre
4. Application mobile (téléchargement)
5. Footer réseaux sociaux

## Features notables
- ✅ **Chatbot Wikit.ai** : bouton flottant — rare sur les mairies, très utile
- ✅ Moteur de recherche intra-site
- ✅ Skip-link + focus states (accessibilité)
- ✅ 5 réseaux sociaux (FB, Insta, X, LinkedIn, YouTube)
- ❌ Pas de multilingue

## Points faibles
- Typographie basique : système fonts, peu de personnalité
- Quasi une seule hero en homepage (peu de variété visuelle)
- Design standard sans microinteractions
- Palette bleu/gris classique sans surprise

## Note globale
7/10 — Bon équilibre entre fonctionnalité et lisibilité. Le chatbot Wikit.ai est une feature notable. Navigation "Mes services" = ton citoyen pertinent. Design propre mais pas mémorable.
