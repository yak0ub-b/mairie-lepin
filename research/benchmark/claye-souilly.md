# Claye-Souilly.fr — Analyse design

## Navigation
- 6 items : Municipalité / Claye toutes générations / Cadre de vie / Loisirs / Démarches / "Je suis"
- Sous-menus étendus (Cadre de vie : 8 sous-items)
- Persona "Je suis" : Ado, Étudiant, Parent, Sénior

## Hero
- Carrousel d'images full-width (photos événementielles et administratives)
- **Pas de CTA explicite** — navigation par carousel uniquement

## Couleurs
- **Primaire** : `#ff5873` (rose vif) — omniprésent sur filtres, boutons, headings
- **Secondaire** : `#333333` (gris-noir) — texte principal
- **Fond** : `#ffffff` + `#f5f4f4` (gris très clair)
- **Accents** : `#8c8c8c`, `#6d6d6d` (gris moyens)

## Typographies
- Non déclarée explicitement (thème WordPress)
- Polices système probable

## Sections homepage (ordre)
1. Header + nav
2. Carousel bannière
3. **"En 1 clic"** (8 liens rapides : carte, portail famille, menus cantine...)
4. Actualités (3 articles en grid)
5. Agenda (6 événements)
6. Magazine (publications PDF)
7. Pratique (5 liens services)
8. Fil social Facebook intégré
9. Footer (horaires, contact, réseaux)

## Features notables
- ✅ Accessibilité : lien dédié + boutons contraste/inversion
- ✅ Flux Facebook intégré (Flow-Flow plugin)
- ✅ Recherche présente
- ✅ RGPD : banneau cookies configuré
- ❌ Pas de chatbot
- ❌ Pas de multilingue

## Points faibles
- Rose `#ff5873` omniprésent → homogénéité lassante, peu professionnelle pour une mairie
- Carrousel sans CTA → appel à l'action manqué
- Hiérarchie typographique faible
- Logo peu mémorable pour une collectivité
- Dépendance WordPress visible (peu de personnalisation fine)

## Note globale
5/10 — Structure correcte ("En 1 clic" bien positionné), mais le rose dominant est un faux pas pour une mairie. Commune voisine directe : Le Pin peut clairement faire mieux.
