# Mitry-Mory.fr — Analyse design

## Navigation
- 6 items : Ma ville / Une ville à vivre / Une ville durable / Une ville qui protège / Vous êtes / Mon compte citoyen
- Navigation "Une ville..." → branding thématique engagé
- Personas "Vous êtes" : jeune, famille, nouveau Mitryen, senior, entreprise, commerçant

## Hero
- Carrousel actualités (3 slides)
- Images larges (1110×362px) + titre + accès direct
- CTA implicites (liens texte vers articles)

## Couleurs
- Blanc/gris clair (fond)
- Bleu/cyan (liens, éléments interactifs)
- Hex non exposés dans CSS inline

## Typographies
- Non spécifiée, thème WordPress standard

## Sections homepage (ordre)
1. Header + nav
2. Bannière actualités (carousel)
3. "L'actualité" (4 articles en grille)
4. "L'agenda" (4 événements)
5. Moteur recherche événements (filtres catégories/lieux/dates)
6. "Le kiosque" (publications PDF)
7. "Documents utiles" + "En photos"
8. **"En 1 clic"** (10 raccourcis : centre santé, portail familles, billetterie...)
9. **"Près de chez vous"** (carte interactive + infos travaux)
10. Footer

## Features notables
- ✅ Recherche avec filtres (catégories/lieux/dates)
- ✅ **Carte interactive** : géolocalisation équipements + travaux
- ✅ **Portail citoyen** "Mon compte citoyen"
- ✅ Applications mobiles (App Store + Google Play)
- ✅ Flux RSS
- ✅ Réseaux : FB + YouTube
- ❌ Pas de chatbot

## Points faibles
- ~50 liens visibles above-the-fold → surcharge informationnelle
- "En 1 clic" placé trop bas (10e section !)
- Carousels sans indicateurs progression
- Carte interactive charge tardive (JS requis)
- Palette monotone sans accent distinctif
- Mobile probablement dense et difficile

## Note globale
6/10 — Bonne richesse fonctionnelle (carte interactive, portail citoyen, recherche filtrée). Navigation branding "Une ville..." bien pensée. Mais surcharge et organisation contestable.
